Terraform Provider Network Mirror

This is a Terraform provider network mirror for terraform-provider-postgresql.

Usage

To use this mirror, add the following to your Terraform CLI configuration file:

  • Linux/macOS: ~/.terraformrc
  • Windows: %APPDATA%\terraform.rc
provider_installation {
  network_mirror {
    url = "https://swile.github.io/terraform-provider-postgresql/"
    include = ["registry.terraform.io/cyrilgdn/postgresql"]
  }
}

Using the Provider

In your Terraform configuration:

terraform {
  required_providers {
    postgresql = {
      source  = "cyrilgdn/postgresql"
      version = "~> 1.0"
    }
  }
}

provider "postgresql" {
  # Configuration options
}
